NOOX AI Device - Host Agent
===========================

This USB disk contains the NOOX Host Agent program.

Files:
------
- noox-host-agent.exe : Windows host agent (auto-launch capable)

Installation:
-------------
1. Copy noox-host-agent.exe to your computer (e.g., C:\NOOX\)
2. Run it with: noox-host-agent.exe --wifi-status=disconnected
3. The agent will connect to your NOOX device via USB serial port

For more information, visit:
https://github.com/NOOX-Device

Note: This disk is stored on the FFat partition of the ESP32-S3.
